# Icône du Module Dorevia Billing CORE

## 📍 Emplacement

L'icône du module doit être placée ici :
```
static/description/icon.png
```

## 📋 Spécifications

- **Format** : PNG
- **Taille recommandée** : 128x128 pixels ou 256x256 pixels
- **Nom du fichier** : `icon.png` (obligatoire)

## ✅ Utilisation

Odoo détecte automatiquement l'icône si elle est dans `static/description/icon.png`.

L'icône apparaîtra :
- Dans la liste des modules (`Apps`)
- Dans le menu principal (si `application=True`)
- Dans les notifications et alertes

## 🎨 Description de l'icône

L'icône représente :
- Un document bleu clair avec un signe dollar ($) blanc
- Un cadenas bleu foncé avec un trou de serrure blanc
- Fond noir

Cette icône symbolise la **sécurité financière** et la **protection des données de facturation**.

---

**Note** : Après avoir ajouté l'icône, mettez à jour le module dans Odoo pour que l'icône soit visible.
