# 📖 Guide Simple : Se Connecter à Odoo et Installer le Module

## 🎯 Étape 1 : Se Connecter à Odoo

### 1.1 Ouvrez votre navigateur

Allez à : **https://odoo.lab.core.doreviateam.com**

### 1.2 Sélectionnez la base de données

Vous devriez voir une page avec :
- Le logo Odoo
- Une liste de bases de données disponibles

**Cliquez sur** : `dorevia_core` (dans la liste)

### 1.3 Connectez-vous

Remplissez le formulaire de connexion :
- **Email** : `admin`
- **Password** : (le mot de passe que vous avez défini lors de la création de la base)

**Cliquez sur** : `Connexion` (ou `Login`)

---

## 🎯 Étape 2 : Installer le Module Dorevia Billing CORE

### 2.1 Accédez aux Applications

Une fois connecté, vous êtes dans l'interface Odoo.

1. **Cliquez sur** le menu `Apps` (ou `Applications`) en haut de la page
   - Si vous ne voyez pas le menu, cliquez sur l'icône ☰ (menu hamburger) en haut à gauche

### 2.2 Recherchez le Module

1. Dans la barre de recherche en haut de la page `Apps`, **tapez** : `Dorevia Billing`
2. Le module **"Dorevia Billing CORE"** devrait apparaître dans les résultats

### 2.3 Installez le Module

1. **Cliquez sur** le module "Dorevia Billing CORE"
2. **Cliquez sur** le bouton `Installer` (ou `Install`)
3. **Attendez** la fin de l'installation (peut prendre 1-2 minutes)

---

## 🎯 Étape 3 : Vérifier l'Installation

### 3.1 Vérifiez que le Module est Installé

1. Retournez dans `Apps`
2. Recherchez "Dorevia Billing CORE"
3. Le statut devrait être **"Installé"** (et non "À installer")

### 3.2 Vérifiez le Menu

1. Dans le menu principal d'Odoo, vous devriez voir **"Dorevia Billing"**
2. Si vous le voyez, c'est que le module est bien installé ! ✅

---

## ❓ Si Vous Ne Voyez Pas le Menu "Apps"

1. **Cliquez sur** l'icône ☰ (menu hamburger) en haut à gauche
2. **Cherchez** "Apps" ou "Applications" dans le menu
3. **OU** allez directement à : `https://odoo.lab.core.doreviateam.com/web#action=base.action_view_base_module_upgrade_install`

---

## ❓ Si Vous Êtes Toujours sur la Page de Sélection de Base

Si vous voyez toujours la page avec la liste des bases de données :

1. **Cliquez sur** `dorevia_core` dans la liste
2. **Connectez-vous** avec :
   - Email : `admin`
   - Password : (votre mot de passe)
