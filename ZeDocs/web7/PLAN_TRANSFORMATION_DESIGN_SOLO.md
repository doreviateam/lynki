# 🎨 Plan de Transformation Design — Inspiration Solo.ai

**Date :** 2026-01-22  
**Objectif :** Transformer le site Dorevia-Vault pour atteindre un niveau de design équivalent à Solo.ai

---

## 📊 Analyse Solo.ai

### Points forts identifiés

1. **Espacement généreux**
   - Marges entre sections : ~120px
   - Padding interne des cartes : ~40px
   - Gap entre éléments : ~24px

2. **Hiérarchie typographique claire**
   - Titres H1 : ~48px, bold
   - Titres H2 : ~36px, bold
   - Sous-titres : ~20px, medium
   - Corps : ~16px, regular

3. **Cartes modernes**
   - Bordures arrondies : 16-20px
   - Ombres douces et subtiles
   - Fond blanc pur ou très léger
   - Hover effects discrets

4. **Icônes proéminentes**
   - Taille : 48-64px
   - Couleurs vives mais harmonieuses
   - Centrées dans les cartes

5. **Couleurs**
   - Fond principal : blanc pur (#FFFFFF)
   - Texte principal : gris foncé (#1F2937)
   - Accent : bleu vif (#3B82F6 ou similaire)
   - Texte secondaire : gris moyen (#6B7280)

6. **Respiration visuelle**
   - Sections espacées de 80-120px
   - Contenu centré avec max-width
   - Beaucoup d'espace blanc

---

## 🎯 Transformations à appliquer

### 1. Espacement global
- [ ] Augmenter padding sections : 80px → 120px
- [ ] Augmenter gap entre cartes : 20px → 32px
- [ ] Augmenter padding cartes : 24px → 40px
- [ ] Augmenter margin-bottom titres : 16px → 32px

### 2. Typographie
- [ ] Augmenter taille H1 : 36px → 48px
- [ ] Augmenter taille H2 : 28px → 36px
- [ ] Augmenter taille sous-titres : 16px → 20px
- [ ] Augmenter taille corps : 14px → 16px
- [ ] Améliorer line-height : 1.5 → 1.6-1.7

### 3. Cartes
- [ ] Fond plus clair : rgba(255,255,255,0.06) → rgba(255,255,255,0.12)
- [ ] Bordures plus arrondies : 18px → 24px
- [ ] Ombres plus douces et subtiles
- [ ] Padding interne : 24px → 40px
- [ ] Hover effects : translateY(-4px) + shadow

### 4. Icônes
- [ ] Taille : 48px → 64px
- [ ] Centrage amélioré
- [ ] Espacement avec texte : 16px → 24px

### 5. Couleurs
- [ ] Contraste amélioré pour texte
- [ ] Couleur accent plus vive
- [ ] Fond de section plus neutre

### 6. Layout
- [ ] Max-width contenu : 1200px
- [ ] Centrage amélioré
- [ ] Espacement vertical entre sections
