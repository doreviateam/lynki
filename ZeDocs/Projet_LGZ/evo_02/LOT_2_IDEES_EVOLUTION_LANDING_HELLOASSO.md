# Lot 2 - Idees d'evolution de la landing HelloAsso

## Objet

Cette note capitalise quelques pistes d'amelioration pour un lot 2 leger de la landing HelloAsso.

Le lot 1 est considere comme satisfaisant :

- la page est plus lisible ;
- le contexte de societe active est explicite ;
- l'orientation metier est meilleure ;
- le niveau de complexite est reste raisonnable.

L'objectif du lot 2 n'est pas de relancer une refonte, mais d'apporter quelques ameliorations ciblees de confort et de lisibilite.

---

## 1. Mieux mettre en valeur le statut

Le statut global remplit bien son role, mais il pourrait etre plus visible visuellement.

Piste :

- afficher le statut sous forme de badge ou pastille plus nette ;
- distinguer plus clairement les cas `production`, `essai` et `configuration a completer`.

Objectif :

- rendre l'etat global lisible en un coup d'oeil.

---

## 2. Rendre les cartes plus actionnables

Les cartes de synthese sont plus claires qu'avant, mais elles restent encore statiques.

Piste :

- rendre chaque carte cliquable ;
- ou ajouter un acces direct discret vers la vue correspondante.

Objectif :

- faire de la landing un veritable point d'entree d'usage quotidien.

---

## 3. Alleger encore le bas de page

Le bloc `Points a verifier` est utile, mais il pourra etre encore mieux hierarchise.

Pistes :

- renforcer visuellement les alertes utiles ;
- garder `Aide rapide` plus discret ;
- reduire encore le texte si les usages deviennent familiers.

Objectif :

- privilegier le signal utile plutot que la lecture.

---

## 4. Conserver le cadre

Pour la suite, il est recommande de conserver les principes poses par le lot 1 :

- landing HelloAsso par societe active ;
- vocabulaire metier simple ;
- structure sobre ;
- pas de refonte lourde sans besoin utilisateur explicite.

---

## Synthese

Le lot 2 pourra donc rester tres leger.

Les trois priorites proposees sont :

1. mieux presenter le statut ;
2. rendre les cartes plus vivantes ;
3. alleger encore le bas de page.

Cela permettrait d'ameliorer la qualite d'usage sans remettre en cause le travail deja stabilise dans le lot 1.
