## ⏳ Processus de Vaulting Automatique

Le vaulting se fait **automatiquement** via le CRON Odoo qui s'exécute **toutes les 5 minutes**.

### Étapes

1. ✅ **Facture validée** : La facture est en statut `posted`
2. ⏳ **CRON #1** : Envoi vers DVIG (toutes les 5 min)
   - Le CRON sélectionne les factures avec `dorevia_vault_status = 'todo'`
   - Envoie l'événement `invoice.posted` à DVIG
   - Met à jour le statut à `pending_proof`
3. ⏳ **Worker DVIG** : Traitement et envoi vers Vault
   - DVIG reçoit l'événement
   - Le worker traite et envoie à Vault avec toutes les métadonnées
4. ⏳ **Vault** : Stockage avec extraction automatique des données
   - Vault reçoit l'événement
   - **Extraction automatique** des 8 champs de facturation :
     - `invoice_number` ← `move_name`
     - `move_type` ← `move_type`
     - `invoice_date` ← `invoice_date`
     - `total_ht` ← `amount_untaxed`
     - `total_ttc` ← `amount_total`
     - `currency` ← `currency_name`
     - `seller_vat` ← `seller_vat`
     - `buyer_vat` ← `buyer_vat`
5. ⏳ **CRON #2** : Récupération de la preuve (toutes les 5 min)
   - Le CRON récupère la preuve depuis Vault
   - Met à jour le statut à `vaulted`
   - Stocke les informations (SHA256, Vault ID, Ledger Hash, JWS)

**Temps estimé** : 5-10 minutes maximum
