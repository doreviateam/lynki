# SPEC_LOGO_DOREVIA_LINKY_v2.1

**Orientation :** Modernité maîtrisée
**Date :** 15 février 2026
**Statut :** Recommandation d'évolution visuelle contrôlée

---

# 0. Objectif stratégique

Positionner Dorevia Linky comme :

* Une infrastructure financière sérieuse
* Un cockpit décisionnel moderne
* Un produit SaaS premium
* Fondé sur la vérité financière scellée

La modernité ne doit jamais compromettre :

* La lisibilité
* L'autorité
* La stabilité
* La perception de conformité

---

# 1. Architecture inchangée (2 niveaux)

```
DOREVIA Linky
BUILT ON SEALED FINANCIAL TRUTH
```

Aucune icône.
Aucun pictogramme.
Aucun élément décoratif supplémentaire.

---

# 2. Modernité maîtrisée — Ajustements autorisés

## 2.1 Typographie

### DOREVIA

* Uppercase
* Tracking +0.18em
* Poids 500
* Couleur Slate-400 (#94A3B8)

### Linky

* Poids 600 (semibold)
* Blanc pur #FFFFFF
* Tracking -0.01em (léger resserrement pour modernité)
* Kerning optique activé

Effet recherché :
Un rendu plus contemporain sans effet flashy.

---

# 3. Accent moderne subtil

Au lieu d'un gradient ou glow :

Autorisé :

* Accent micro-contraste sur le "L" de Linky
* Ou légère variation de teinte sur la lettre "y"

Exemple discret :
Linky → blanc pur
Lettre "y" → blanc légèrement froid (#F1F5F9)

Variation max 5 % de contraste.

Objectif :
Créer une signature visuelle sans tomber dans le marketing.

---

# 4. Hover moderne maîtrisé

Autorisé :

* brightness(1.04)
* transition 160ms ease-out
* aucune transformation scale
* aucune ombre

Option premium :

* underline ultra fin (1px) en Emerald-400
* apparition en 120ms
* uniquement au survol

Pas d'animation agressive.

---

# 5. Tagline — Modernisation subtile

Texte :

BUILT ON SEALED FINANCIAL TRUTH

Modernisation autorisée :

* Opacité 90 %
* Tracking +0.22em
* Poids 500
* Couleur Emerald-400 mais désaturée à 85 %

**Règle d'affichage :**
* Toujours sur **1 ligne** (`whitespace-nowrap`)
* Masquée si la largeur du device rend la lecture difficile pour un humain
* Implémentation : affichage à partir de 768px (md breakpoint) uniquement

Objectif :
Accent vivant mais institutionnel.

---

# 6. Fond navbar

Fond recommandé :

#0F172A

Modernité autorisée :

* Subtil border-bottom 1px rgba(255,255,255,0.04)
* Pas de gradient
* Pas de glassmorphism

---

# 7. Micro-détails premium

Autorisé :

* Subpixel antialiasing optimisé
* Letter-spacing cohérent sur tout l'écosystème
* Alignement pixel perfect

Interdit :

* Neon
* Ombres colorées
* Effets 3D
* Blur

---

# 8. Équilibre recherché

Le logo doit se situer ici :

Autorité ←—————●—————→ Startup flashy
Modernité maîtrisée

Linky doit paraître :

* Technologique
* Fiable
* Stable
* Contemporain

Jamais :

* Excité
* Fun
* Décoratif

---

# 9. Résumé exécutif

Modernité apportée par :

* Micro-contrastes typographiques
* Ajustements de tracking
* Hover subtil
* Accent maîtrisé

Autorité conservée par :

* Absence d'icône
* Absence de gradient
* Couleurs sobres
* Hiérarchie stricte
