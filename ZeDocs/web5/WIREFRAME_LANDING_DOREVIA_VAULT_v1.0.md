## 📐 Responsive — Breakpoints

### Mobile (< 768px)

**Hero** :
- Headline sur 2 lignes max
- Sous-titre raccourci : "Pilotez avec des données fiables, sécurisées et vérifiables."
- Visual masqué ou empilé sous le texte
- CTAs empilés verticalement

**Sections** :
- Cartes en 1 colonne (empilées)
- Texte centré
- Espacement généreux entre sections

### Tablette (768px - 991px)

**Hero** :
- Layout 2 colonnes possible
- Visual réduit mais visible

**Sections** :
- Cartes en 2 colonnes
- Texte aligné à gauche

### Desktop (≥ 992px)

**Hero** :
- Layout 2 colonnes complet
- Visual à droite, texte à gauche

**Sections** :
- Cartes en 3-4 colonnes selon section
- Espacement optimal

## 🎨 Éléments visuels recommandés

### Couleurs

- **Primaire** : #2f57d7 (bleu Dorevia)
- **Secondaire** : #1e40af (bleu foncé)
- **Accent** : #fbbf24 (jaune pour CTAs)
- **Texte** : #1e293b (gris foncé)
- **Texte secondaire** : #64748b (gris moyen)

### Typographie

- **Headline** : 48-56px, bold (desktop) / 32-40px (mobile)
- **Sous-titre** : 20-24px, normal
- **Corps** : 16-18px, normal
- **CTA** : 18px, bold

### Espacements

- **Entre sections** : 80px (desktop) / 60px (mobile)
- **Padding sections** : 80px vertical (desktop) / 60px (mobile)
- **Gap cartes** : 24px (desktop) / 16px (mobile)
