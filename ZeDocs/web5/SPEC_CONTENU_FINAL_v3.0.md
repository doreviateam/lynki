## 14. Messages formulaires

### Formulaire de contact

**Titre** :
```
Parlez-nous de votre projet
```

**Champs** :
- Nom : `Votre nom`
- Email : `Votre email`
- Entreprise : `Votre entreprise` (optionnel)
- Message : `Votre message`

**Placeholders** :
- Nom : `Jean Dupont`
- Email : `jean.dupont@entreprise.fr`
- Entreprise : `Mon Entreprise SARL`
- Message : `Décrivez votre besoin...`

**Bouton submit** :
```
Envoyer
```

**Messages validation** :

**Succès** :
```
Merci ! Votre message a été envoyé. Nous vous répondrons dans les plus brefs délais.
```

**Erreur** :
```
Une erreur est survenue. Veuillez réessayer ou nous contacter directement.
```

**Erreurs champs** :
- Nom vide : `Le nom est requis`
- Email invalide : `Veuillez entrer un email valide`
- Message vide : `Le message est requis`
- Message trop court : `Le message doit contenir au moins 10 caractères`
