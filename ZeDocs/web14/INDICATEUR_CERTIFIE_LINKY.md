# Indicateur « Certifié » dans Linky — Règle d’affichage

**Principe** : Le **pourcentage Certifié** est toujours une valeur **globale** (toutes périodes). Il n’est **pas recalculé** en fonction de la période sélectionnée.

- **Certifié : X %** = (nombre de factures vaultées sur l’ensemble des données) / (nombre de factures postées sur l’ensemble des données). Ce ratio reflète la maturité globale du vaulting (ex. 100 % si toutes les factures postées sont vaultées).
- **X factures concernées** = nombre de factures (vaultées) **sur la période affichée** dans la carte. Ce chiffre change avec le filtre période et décrit les documents qui portent les montants affichés.

Quel que soit le filtre période (Mois en cours, Trimestre, etc.), l’utilisateur voit donc :
- le **même** indicateur de certification (ex. Certifié : 100 %) ;
- le **nombre de factures concernées** par la période (ex. 6 factures concernées pour le mois).
